## 💻 Sobre a atividade

Criar uma página que demonstre a utilização de:
- fundos de caixas em “cor sólida”, “cor sólida com transparência”, “gradiente de cores”, “imagem única”, “imagem repetida”;
- estilos de bordas, isto é, utilizando tipos, cores, espessuras e arredondamentos de bordas;
- efeito box-shadow em uma imagem ou foto no formato de um círculo.
